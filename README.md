# 极客狐尼克 - 智能语音桌面小车

这是一个基于ESP32的智能小车控制系统，集成了时钟显示、语音控制、网页控制、表情动画等多种功能。

## 📋 功能特性

### 🚗 智能小车模式
- **方向控制**：前进、后退、左转、右转、停止
- **表情系统**：10种动态表情（开心、生气、困倦、大笑、困惑、流汗、惊讶、困倦、无语、默认）
- **动作指令**：跳舞、转圈、持续摇摆、摇摆舞
- **随机模式**：睡眠模式、摆动模式、好奇模式
- **LED特效**：警灯红蓝闪烁、氛围灯呼吸效果

### ⏰ 时钟模式
- 网络时间同步（NTP）
- WiFi连接状态显示
- 日期、时间实时显示

### 🎤 语音控制（天问Block）
- 基础移动：前进、后退、左转、右转、停止
- 模式切换：时钟/小车模式
- 动作控制：跳舞、转圈、摇摆
- 特效控制：警灯、氛围灯、关灯
- **隐藏彩蛋**：特定语音指令触发隐藏信息

### 🌐 网页控制
- 手机/电脑通过浏览器访问控制面板
- 响应式设计，触摸优化
- 实时指令响应

## 🔧 硬件要求

### 核心部件
- ESP32 开发板（推荐使用 ESP32-WROOM-32）
- OLED 显示屏（128x64，I2C接口）
- 电机驱动模块（L298N 或类似）
- 直流电机 x2（带轮子）
- RGB LED 红蓝各一个（或双色LED）

### 引脚连接

| 组件 | 引脚 | 说明 |
|------|------|------|
| 左轮正转 | GPIO 0 | LF |
| 左轮反转 | GPIO 1 | LB |
| 右轮正转 | GPIO 2 | RF |
| 右轮反转 | GPIO 3 | RB |
| 红色LED | GPIO 5 | PWM |
| 蓝色LED | GPIO 7 | PWM |
| I2C SDA | GPIO 9 | OLED数据 |
| I2C SCL | GPIO 8 | OLED时钟 |
| UART1 RX | GPIO 20 | 语音模块接收 |
| UART1 TX | GPIO 21 | 语音模块发送 |

## 🚀 快速开始

### 1. 首次上电
- ESP32会创建一个名为 **"极客狐尼克"** 的WiFi热点
- 使用手机连接该热点（无密码）
- 浏览器访问 `192.168.4.1`

### 2. 配置WiFi（可选）
- 点击右上角 **"⚙️ 配网"** 按钮
- 连接到家中路由器，实现远程控制

### 3. 使用控制面板
- **时钟模式**：显示网络时间，电机锁死
- **小车模式**：启用所有控制功能
  - 方向按钮：按住持续移动
  - 动作指令：跳舞、转圈等
  - 表情系统：10种动态表情
  - 随机模式：设置小车的随机行为

### 4. 语音控制（天问Block）
连接天问Block语音模块到UART1，支持以下指令：

| 指令 | 功能 | 指令 | 功能 |
|------|------|------|------|
| go | 前进 | back | 后退 |
| left | 左转 | right | 右转 |
| stop | 停止 | dance | 跳舞 |
| spin | 转圈 | swing | 持续摇摆 |
| car | 小车模式 | clock | 时钟模式 |
| police | 警灯模式 | ambient | 氛围灯 |
| gd | 关灯 | reconfig | 重新配网 |

### 5. 隐藏彩蛋 🌟
连续说出特定指令可触发隐藏信息：
- 说 **"clock"** 3次
- 说 **"off"** 7次（小车模式下）
- 说 **"nike"** 3次（小车模式下）
- 说 **"ak"** 3次（小车模式下）
- 说 **"fm"** 9次（小车模式下）

## 📱 网页界面功能

### 方向控制
- 圆形方向按钮，支持按住持续移动
- 独立全局停止按钮

### 随机模式
- 💤 睡眠：无随机动作
- 🔄 摆动：轻柔随机动作
- 👀 好奇：更多随机表情和动作

### 动作指令
- 💃 跳舞（前后摇晃→转圈→再次摇晃→快速转圈）
- 🔄 转圈（持续10秒，中途切换表情）
- 🎵 持续摇摆（2秒来回摇摆）
- 💫 摇摆舞（28秒摇摆舞蹈）

### 表情控制（10种）
默认、开心、生气、疲惫、大笑、困惑、流汗、惊讶、困倦、无语

## 🎮 系统架构

### 状态机设计
- **设备模式**：时钟模式 / 小车模式
- **动作状态**：无动作 / 跳舞 / 转圈 / 摇摆 / 摇摆舞 / 警灯 / 氛围灯
- **随机模式**：关闭 / 轻柔 / 正常

### 核心特性
- ✅ 非阻塞编程（所有动作基于`millis()`）
- ✅ 实时打断（新指令立即停止当前动作）
- ✅ 电机锁死保护（时钟模式自动锁死电机引脚）
- ✅ 语音指令计数系统（彩蛋触发机制）

## 🐛 常见问题

### OLED不显示
- 检查I2C地址（默认为0x3C）
- 确认SDA/SCL引脚连接正确（GPIO9/GPIO8）
- 检查电源电压（OLED需要3.3V）

### 电机不转
- 检查电机驱动电源
- 确认电机引脚正确（GPIO0-3）
- 时钟模式下电机自动锁死，切换到小车模式

### 无法配网
- 长按复位按钮重新启动
- 使用语音指令"reconfig"强制重新配网
- 手动清除WiFi配置后重启

### 语音无响应
- 检查天问Block模块电源
- 确认UART1接线（RX20/TX21）
- 检查串口波特率（9600）
